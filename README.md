# Python学生就业管理系统 python888

## 项目概述

本项目是基于Python的学生就业管理系统，旨在帮助学院、企业和学生实现就业信息的有效管理。系统包括学院、专业、企业、岗位、管理员、学生等多方面信息的管理功能。

## 技术栈

- **前端：** HTML、CSS、JavaScript、jQuery
- **后端：** Python、Django
- **数据库：** MySQL
- **开发工具：** PyCharm、Navicat

## 运行环境

- **Python 版本：** 3.0及以上
- **MySQL 版本：** 8.0

## 功能模块

- **学院信息管理：** 管理学院相关信息
- **专业信息管理：** 管理专业相关信息
- **企业信息管理：** 管理企业相关信息
- **岗位信息管理：** 管理岗位相关信息
- **管理员信息管理：** 管理管理员相关信息
- **学生信息管理：** 管理学生基本信息
- **学生教育经历：** 管理学生的教育背景
- **学生项目经历：** 管理学生的项目经历
- **学生投递记录：** 跟踪学生的岗位投递记录
- **个人信息修改：** 允许用户修改个人信息和密码

## 系统角色

- **管理员：** 负责系统的维护和用户管理
- **学生：** 可以查看、投递岗位，管理个人信息
- **教师/企业：** 可以发布岗位，管理相关信息

1. 确保Python 3.0及以上和MySQL 8.0环境已安装。
2. 使用PyCharm或命令行创建虚拟环境。
3. 克隆项目代码到本地。
4. 安装依赖包。
5. 运行数据库迁移命令。
6. 启动Django应用。
